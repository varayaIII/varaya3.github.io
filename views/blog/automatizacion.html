Pipelines de CI/CD: La Línea de Ensamblaje

Un pipeline de CI/CD es una línea de ensamblaje automatizada...

  1. Build: Se compila el código y, más comúnmente hoy en día, se construye una imagen de contenedor (Docker)...
  2. Test: Se ejecutan pruebas automatizadas contra el artefacto recién creado...
  3. Seguridad: ¡La seguridad no es negociable!...
  4. Despliegue: Si todas las etapas anteriores (o gates de calidad) se superan, el pipeline despliega el artefacto...
